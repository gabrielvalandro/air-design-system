How to install

Here you can quickly introduce the ways of installing it in your user's project. The options shown below are optional and completely up to you.

CDN

Here you will provide the URL for the files of your design system. You could also advice your users to place the script tag before the body closing tag due to performance.

					<link rel="stylesheet" href="https://airdesignsystem.com/download/air-design-system.min.css">
				
					<script src="https://airdesignsystem.com/download/air-design-system.min.js"></script>

Source code

If you wish to give your user more control over the code of your framework you can make the source code downloadable

NPM

Downloading your framework via NPM, the node package manager

					npm install air-design-system

Bower

Downloading your framework via bower

					bower install air-design-system
